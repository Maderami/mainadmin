Во вложении articles.html представлена статическая страница со списком статей. Разработать на её основе страницу с редактируемым контентом: статью должны выводиться из инфоблока в административной панели "1С Битрикс: Управление сайтом".
Поля статьи в административной панели будут следующими:
- Дата
- Заголовок
- Краткое описание
- Изображение
Произвести полную интеграцию представленного дизайна с компонентом Битрикс, включая вывод даты, картинок, постраничной навигации.

Для разработки можно использовать пробную версию Битрикс. Выполненное задание предоставить в виде полной резервной копии сайта, созданной встроенными средствами Битрикс. Шифрование резервной копии в данном случае необязательно и не приветствуется.
